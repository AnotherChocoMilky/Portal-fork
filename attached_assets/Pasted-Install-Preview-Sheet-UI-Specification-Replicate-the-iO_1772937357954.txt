Install Preview Sheet UI Specification

Replicate the iOS App Store install preview modal sheet design. The sheet should appear as a rounded floating card over a blurred background, centered near the middle of the screen.

This UI should be implemented inside:
	•	InstallPreviewView.swift
	•	InstallProgressView.swift

The component should visually match the App Store install prompt but with the modified fields described below.

⸻

Sheet Layout

Container
	•	Full width modal card
	•	Rounded corners around 28–32pt
	•	Dark blurred background (system material)
	•	Slight drop shadow
	•	Appears over dimmed or blurred background content
	•	Padding around content: 20–24pt

⸻

Header Row

Top horizontal row.

Left

Text title:

Install App

Font:
	•	.system(size: 22, weight: .bold)

Color:
	•	Primary label color (white in dark mode)

Right

Circular close button.

Design:
	•	Grey translucent circle
	•	Inside icon: xmark
	•	Size: ~36–40pt
	•	Tap action: dismiss the sheet

⸻

App Information Card

Below the header place a rounded rectangular card containing app metadata.

Layout

Horizontal stack:

Left side:
App icon

Right side:
App information

Spacing between icon and text: 12–14pt

Card styling:
	•	Rounded corners ~18pt
	•	Background: secondary system fill
	•	Padding: 14–16pt

⸻

App Icon

Square icon.

Size:

60x60

Corner radius:

14


⸻

App Metadata Text

Vertical stack.

Line 1

App name + age badge.

Example layout:

Golf Piko   4+

Implementation:
	•	App name
	•	.headline
	•	Bold
	•	Age rating
	•	Small rounded badge
	•	Grey background
	•	Font .caption
	•	Padding horizontal: 6
	•	Padding vertical: 2

Line 2

Install Button

Centered large button.

Label:

Install

Button styling:
	•	Blue system button
	•	Capsule shape
	•	Height: 44–48pt
	•	Width: ~200pt
	•	Font: .headline
	•	Text color: white

Tap action:

start installation
transition to InstallProgressView


⸻

InstallProgressView Behavior

Once installation starts:

Replace the Install button with a progress indicator.

Progress UI:

ProgressView(value: installProgress)

Display:

Installing...

Below progress bar show:

XX%

Example:

Installing...
[progress bar]
45%

When progress reaches 100:

Installed

Button becomes:

Open


⸻

Required Data Fields

InstallPreviewView must receive:

appName
appIcon
bundleID
ageRating
installSource

Install source enum:

enum InstallSource {
    case source(name: String)
    case userImported
}

Display logic:

switch source {
case .source(let name):
    Text(name)

case .userImported:
    Text("User Imported")
}
