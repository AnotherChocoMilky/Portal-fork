iOS/iPadOS App Signer & Installer
Portal

Portal is a powerful iOS/iPadOS app signer and installer that lets you sign, manage, and install applications directly on your device. Built with SwiftUI, it provides a native, privacy-focused experience where all signing happens locally on your device.

Download Latest IPA

Requires iOS/iPadOS device with sideloading capability

Features

Modern Swift Interface

Clean, intuitive UI with support for light, dark, and tinted app icons.

Sign & Install Apps

Sign IPA files using your .p12 certificate and .mobileprovision profile.

Multiple Installation Methods

Local server installation or pairing-based installation via VPN.

Certificate Management

Import, view, and manage multiple certificates with PPQ check status.

Tweak Injection

Inject .dylib and .deb files using ElleKit with configurable paths.

Privacy First

No tracking, no analytics. Everything happens on your device.

About This Project

This is a native iOS/iPadOS application built with Swift and SwiftUI. It requires Xcode on macOS to build and can only run on iOS devices.

The source code is available on GitHub.
